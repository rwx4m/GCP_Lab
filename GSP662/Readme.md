## Hosting Web Apps on Google Cloud Using Compute Engine
---

REMEMBER, you only need to run ZONE, no need to run REGION!!
```bash
gcloud config set compute/zone "<your_zone>"
export ZONE=$(gcloud config get compute/zone)
```
